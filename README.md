# To-Do List Project

This is a simple To-Do List project that allows users to add, delete, and update tasks.

## Features

- Add a new task
- Delete an existing task
- Update an existing task

## Technologies Used

- Django: 5.0
- Python:  3.11.2
- HTML
- CSS
